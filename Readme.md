# LanEnum

Simple script to enumerate PTR DNS records pointing to internal IPs on a LAN.

## Getting Started

### Dependencies

```shell
pip install -r requirements.txt
```

### Emumerate PTR DNS records

```shell
$ chmod +x lanenum.sh && ./lanenum.sh
```
